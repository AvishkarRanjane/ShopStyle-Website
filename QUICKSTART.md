## How to View Your Website

### **Option 1: Direct File Opening (Simplest)**
1. Navigate to: `c:\Users\AVISHKAR\Documents\cloths website`
2. **Double-click** `index.html`
3. Your default browser will open the website
4. Enjoy your professional e-commerce store! 🎉

### **Option 2: Using VS Code**
1. Open VS Code
2. Open the folder: `c:\Users\AVISHKAR\Documents\cloths website`
3. Right-click on `index.html`
4. Select "Open with Live Server" (if installed)
5. Website opens in browser

### **Option 3: Using Browser**
1. Open your browser (Chrome, Firefox, Safari, Edge)
2. Press `Ctrl + O` (or `Cmd + O` on Mac)
3. Navigate to: `c:\Users\AVISHKAR\Documents\cloths website\index.html`
4. Click Open

## 🎨 Customization Tips

### Change Store Name
Edit the logo text "ShopStyle" in all HTML files

### Add Your Logo
Replace the shopping bag icon with your logo image

### Change Colors
Edit `:root` variables in `style.css`:
```css
--primary-color: #667eea;  /* Change primary color */
--secondary-color: #764ba2; /* Change secondary */
```

### Add More Products
Edit `productsDatabase` in `script.js` and add new products

### Update Product Images
Replace Unsplash URLs with your own product images

### Change Prices
Update product prices in `script.js`

### Adjust Discount
Modify discount percentages in `script.js`

### Update Contact Info
Edit contact details in `contact.html`
